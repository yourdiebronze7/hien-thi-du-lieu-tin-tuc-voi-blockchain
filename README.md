# Hệ Thống Hiển Thị Dữ Liệu Tin Tức Với Blockchain

## Giới thiệu
Hệ thống này cho phép người dùng gửi và lưu trữ tin tức lên một mạng phi tập trung sử dụng công nghệ blockchain. Điều này giúp đảm bảo tính minh bạch, bảo mật và xác thực của thông tin. 

## Tính năng
- Gửi tin tức và thông tin lên blockchain.
- Kiểm tra tính xác thực nguồn gốc của tin tức.
- Tạo điều kiện cho người dùng bình luận và thảo luận về tin tức.
- Cung cấp hệ thống xếp hạng cho các bài viết.

## Công nghệ sử dụng
- Blockchain (Ethereum, Hyperledger, hoặc một blockchain tùy chỉnh)
- Frontend: React hoặc Vue.js
- Backend: Node.js hoặc Python

## Cài đặt
1. Clone repository này.
2. Cài đặt các dependencies cần thiết với `npm install` hoặc `pip install`.
3. Chạy ứng dụng với `npm start` hoặc `python app.py`.
